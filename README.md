# Sales Data Analysis using SQL

## Project Overview
This project analyzes sales data using SQL. The database contains customers, products, orders, and order details.

## Tools Used
- MySQL
- SQL
- VS Code

## Database Tables
- Customers
- Products
- Orders
- Order_Details

## Analysis Performed
- Total Revenue
- Top Selling Products
- Sales by City
- Monthly Sales Trend
- Top Customers
